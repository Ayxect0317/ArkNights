アークナイツ　各種データまとめ

本サイトについて

株式会社Youster様が運営されているソーシャルゲーム【アークナイツ】の情報を、
個人的にわかりやすくまとめたサイトです。
作成に当たり、【アークナイツ攻略 Wiki】様 (https://arknights.wikiru.jp/) 様のデータを
引用させていただいております。

閲覧方法について

本サイトはJavascriptを使用しております。
閲覧するにあたり、ローカルサーバーとしてSERVED (http://enjalot.github.io/served/) を使用しております。
SERVEDをダウンロード、起動後、本HTMLファイルをドラッグ&アドロップし、表示されるURLをクリックしてください。
